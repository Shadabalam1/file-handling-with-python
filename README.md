# 📂 File Handling with Python

A simple Python-based CLI script that performs basic file handling operations using CRUD (Create, Read, Update, Delete).

---

## 🚀 Features

* 📄 Create new files
* 📖 Read file content
* ✏️ Update files

  * Rename file
  * Overwrite content
  * Append content
* 🗑️ Delete files

---

## 🛠️ Tech Stack

* Python
* pathlib module

---

## ▶️ How to Run

```bash
git clone https://github.com/Shadabalam1/file-handling-with-python.git
cd file-handling-with-python
python main.py
```

---

## 📌 Usage

Choose from the menu:

* 1 → Create File
* 2 → Read File
* 3 → Update File
* 4 → Delete File
